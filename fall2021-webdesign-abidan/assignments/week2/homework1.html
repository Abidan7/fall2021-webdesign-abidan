Research of two eBooks about HTML and CSS.

First Topic:
    HTML: A Beginner’s Tutorial
    By:  Chris Coremans
    Book:  116 pages
    Publisher:  Brainy Software Corp. © 2015

About the Author
Chris Coremans is software engineer specializing in front-end development.
An expert in HTML5, JavaScript, JQuery and Flash development with ActionScript.
I like his professional background!

About the Book
The author is confident that, in his book, any person who is interested in learning
HTML can do so without resorting to a "1000 page book". Mr. Coremans emphazises
that the reader will learn the basics techniques to style any plain document
and turn it to good looking web page fairly quickly.

My Personal Opinion
I personally like how the book is presented, the examples are clear and easy to follow.
The book has good examples that I was able to follow.
This book is perfect to supplement and practice what we are learning in this cohort
and would definetely recommend it. 

Second Topic:
    CSS: The Missing Manual
    By: David Sawyer McFarland
    Publisher: Pogue Press, 2009

About the Book
This book describes what CSS is, a Cascade style Sheets that gives the user creative control
over the layout and design of his/her web page. The purpose of this book is to give the reader
a better undertanding how CSS enchances HTML formatting with streamlined tags that HTML may have
limits over specific web browsers.
The book quickly points out the need of having basic knowledge of HTML before CSS can be introduced.

My Personal Opinion
This book contents are more like an informational directory rather than a book with tools and examples.
It lacks of engaging materials for the reader to start learning. Also, it seems a somewhat outdated.

The form HTML element

A "form" element is tool used to create a interactive controls or "inputs" for web-based forms
in order to accept data from the user; there are many types of input data and control widgets,
depending on the device and user agent. Note: The input element is one of the most powerful and complex
in all of HTML due to the large number of combinations of input types and attributes.
